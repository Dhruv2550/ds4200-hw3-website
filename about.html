My Latest Project: FAIR

One of my recent projects is building a web application called FAIR (Fast AI-Assisted Investigation and Review). I built this app at my last co-op with the NJ Attorney Generals Office. FAIR is designed to help investigators review and summarize their legal interviews. Investigatiors upload video recordings of interviews and transcripts, and FAIR uses the GPT-4o API to transcribe the audio. Then, it applies natural language processing techniques and specific formats to generate concise summaries and extract key topics from the transcripts. It reduced interview review time from more than an hour to just minutes, allowing investigators to focus on analysis rather than manual review.

Some of the technologies and languages used for the FAIR project were:

  • TypeScript
  • Python
  • HTML
  • CSS
  • Azure OpenAI
  • Azure Static Web Apps
  • Azure PostgreSQL
  • Azure Container Service
